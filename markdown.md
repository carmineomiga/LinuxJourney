# 마크다운 사용법
마크다운(markdown)을 사용하는데 필요한 지식들

## 1. 제목 Headers
- 글머리(제목)의 크기는 1~6까지 존재한다.
- '#'의 개수로 글머리 크기를 조정한다.
	(-,=로도 조정이 가능하긴 하지만, '#'과는 방법이 다르고, 더 불편하다.)
- 표현 방법과 결과는 다음과 같다
```
# This is a H1
## This is a H2
### This is a H3
#### This is a H4
##### This is a H5
###### This is a H6
```
# This is a H1
## This is a H2
### This is a H3
#### This is a H4
##### This is a H5
###### This is a H6
