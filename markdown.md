# Markdown 마크다운 사용법 
마크다운(markdown)을 사용하는데 필요한 지식들을 모아놓은 곳입니다.


***
## 1. Header 제목
- 글머리(제목)의 크기는 1~6까지 존재한다.
- '#'의 개수로 글머리 크기를 조정한다.

	(-,=로도 조정이 가능하긴 하지만, '#'과는 방법이 다르고, 더 불편하다.)
- 표현 방법과 결과는 다음과 같다
```
# This is a H1
## This is a H2
### This is a H3
#### This is a H4
##### This is a H5
###### This is a H6
```
# This is a H1
## This is a H2
### This is a H3
#### This is a H4
##### This is a H5
###### This is a H6


## 2. Task List 체크 박스
- 체크된 박스는 '[x]', 체크되지 않은 박스는 '[ ]'를 사용한다.
- 사용한 바에 따르면 맨 앞에서만 사용이 가능하며,

	중간이나 뒤에서는 사용이 불가능하다.
- 표현 방법과 결과는 다음과 같다.
```
- [x] water
- [ ] water
1. [x] milk
2. [ ] milk
```
- [x] water
- [ ] water
1. [x] milk
2. [ ] milk
