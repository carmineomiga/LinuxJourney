# 1. topic
1. '-','='로 글머리크기(header size)를 조정할 수 있는가?
2. 조정 방법이 '#'과 같은가?

# 2. test
This is a H1
======
This is a H2
=====
This is a H3
====
This is a H4
===
This is a H5
==
This is a H6
=


This is a H1
------
This is a H2
-----
This is a H3
----
This is a H4
---
This is a H5
--
This is a H6
-

# 3. result
1. '-','='로 글머리 크기가 조정 가능하다.
2. '#'과는 조정 방법이 다르다.

# 4. conclusion
글머리 크기를 조정하는 것은 '#'이 가장 편리하다.
